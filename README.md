# Flahtik — Spatial Intelligence Platform

**flahtik.com** — Arctic Blue redesign, March 2025

## Stack
- Pure HTML / CSS / Vanilla JS — zero dependencies
- Fonts: Bricolage Grotesque + Outfit via Google Fonts
- Forms: Formspree
- Hosting: Netlify (auto-deploy from main branch)
- DNS: Netlify DNS → flahtik.com

## Structure
/css          → Design system + per-section styles
/js           → main.js (all interactions)
/sections     → One HTML file per section
/components   → nav.html, footer.html
/assets       → Logo, motif, images

## Sections
1. Hero
2. Services / Solutions
3. Platform / How It Works
4. Sectors
5. Stats / Impact
6. About / Mission
7. Insights / Blog
8. Contact
